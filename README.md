# RPN Калькулятор на С++
Реализация калькулятора с использованием обратной польской нотации (RPN)
Поддержка токенизации, парсинг, обработку унарного минуса и вычисление выражений через стек.

---

## Возможности
- Токенизация чисел, операторов и скобок
- Поддержка унарного минуса
- Преобразование выражения в RPN
- Вычисление выражений
- Поддержка операторов: +, -, *, /, ^
- Сборка через CMake

---

## Пример
- Ввод: 3 + 4 * 2 / (1 - 5) ^ 2
- RPN: 3 4 2 * 1 5 - 2 ^ / +
- Результат: 3.5

---

## Используемые алгоритмы

### Tokenizer - O(n)
- классификация символов
- чтение чисел с проверкой десятичной точки
- корректная обработка унарного минуса

### Parser
- стек операторов
- учет приоритета операторов
- учет ассоциативности
### Evaluator
- стек значений
- выполнение бинарных операций
- выполнение унарного минуса
- сообщение об ошибках при делении на 0 и некорректной RPN

## Сложность

### Временная сложность
- Разбиение на токены (tokenizer) — O(n)
- Преобразование в RPN — O(n)
- Вычисление выражения — O(n)
Итог: полный процесс — O(n)

### Пространственная сложность
- Список токенов — O(n)
- Стек операторов — O(n)
- Стек вычислений — O(n)
Итог: использование памяти — O(n)
